#Servidor Node

##Descrição:

Os comandos a seguir referem-se a instalação de um servidor node, que tem a função de adicionar usuários à uma lista.

##Instalação: 

### - O primeiro passo é o rodar o comando npm install para poder instalar as dependências.
    
 	npm install
### - Logo após, instale o npm em formato global.

 	npm install -g nodemon

### - Feito isso, rode o comando a seguir.

	npm install nodemon --save

### - E por fim, para rodar o servidor execute o comando npm start.

	npm start
